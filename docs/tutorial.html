Queryable Tutorial

Guida rapida in stile homepage per configurare Queryable, annotare le entity e generare le REST API.

Open GitHub repo 
mvn io.quarkus.platform:quarkus-maven-plugin:3.34.6:create \
  -DprojectGroupId=it.queryable \
  -DprojectArtifactId=awesomeproj \
  -Dextensions="jdbc-postgresql,resteasy-jackson,hibernate-orm-panache" \
  -Dpath="/awesomeproj" \
  -DnoCode
Step 1: Setup

Crea il progetto Quarkus e aggiungi il plugin nel pom.

Step 2: Annotazioni

Usa Q, QLike, QList, QLogicalDelete sulle entity.

Step 3: Generazione

Esegui queryable:install e queryable:source.

Esempio filtro aggiornato

if (nn("obj.code")) {
  search.filter("Customer.obj.code", Map.of("code", get("obj.code")));
}
if (nn("like.name")) {
  search.filter("Customer.like.name", Map.of("name", likeParamToLowerCase("like.name")));
}
search.filter("Customer.obj.active", Map.of("active", true));
